# Hikrobot 3D轮廓仪智能选型工具

海康机器人3D轮廓仪在线选型工具，支持参数选型与竞品对标。

## 功能

- 📋 **参数选型**：输入重复精度需求、产品尺寸、景深需求，自动推荐最佳型号
- 🔄 **竞品对标**：支持基恩士/LMI/深视/翌视/光子/西克/华睿/梅卡 8大品牌型号反查
- 🏆 **智能排序**：Z重复精度按X线宽插值计算，精度→景深→线宽优先级排序
- 📱 **响应式设计**：手机/平板/电脑均可使用

## 使用方式

1. 直接浏览器打开 `index.html`
2. 或部署到 GitHub Pages / Gitee Pages / 任意静态服务器
